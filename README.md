# SudokuGenerator
Adeeb Hassan

In this project the task is to created a fully solved 9x9 sudoku puzzle ensuring no duplicates in 3x3s, rows, and  columns of numbers 1-9

To run this program click the green code button in the front page and create codespace. Then go to run and debug on the left side of the screen for it to run

The Sudoku board is generated starting from a randomly genrated sequence of numbers 1-9 that is added and shifted constatly on the board to produce a solved sudoku through looping through 2D array  with 
the values of the board inside of it (See Design document for more information)

The files in this repository are the read.me file, SudokuGenerator.java, and the design document pdf
